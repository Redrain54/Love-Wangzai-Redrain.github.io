React简介

  • React是用于构建用户界面的JavaScript库，构建快速响应的大型 Web 应用程序

  • React 执行的策略是：异步更新——异步可中断的更新

    若在浏览器给定的时间内未执行完成，那么就会中断(准确说是暂停)本次执行工作，并将控制权交给浏览器，React则等待下一帧时间到来继续工作

  • 虚拟DOM

    原因：直接对DOM对象进行操作，例如：在现有DOM树中新增节点，删除节点，或改变当前节点的位置和大小等，将引起浏览器的reflow操作；改变背景颜色，改变字体大小，将引起浏览器的repaint操作。这些操作都会增加耗时，降低浏览器渲染的性能。

  • 组件系统（模块化编程）

    单向数据流 因为单向，所以各种变化都是可预计、可控制的

  • ReactDom.render() 后会发生什么

    1. 调度：产生更新（生成新的VDOM）
    2. 协调：决定需要更新什么组件（通过diff算法）
    3. 渲染：将更新的组件渲染到页面（遍历跟新队列，将所有的变更一次性更新到DOM上）

  • react 在什么情况下会执行render

    1. 初始化 setState
    2. 不论组件的state是否发生变化
    3. setState不会立刻改变React组件中state的值（异步）
      多次setState函数调用产生的效果会合并
    4. shouldComponentUpdate函数返回false，因为更新被中断，所以不调用render，但是React不会放弃掉对this.state的更新的，依然会更新this.state
    5. 父组件重新渲染，每当父组件重新render导致的重传props，子组件将直接跟着重新渲染

React脚手架

  • ×××脚手架：用来帮助程序员快速创建一个基于×××库的模板项目
    • 包含了所有需要的配置（语法检查、jsx编译、devServer……）
    • 下载好了所有相关的依赖
    • 可以直接运行一个简单的效果
  • react提供了一个用于创建react项目的脚手架库：create-react-app
  • 项目的整体技术架构为：react + webpack + es6 +eslint
  • 使用脚手架开发的项目的特点：模块化、组件化、工程化
  • 创建项目并启动
    • 全局安装：打开cmd键入指令npm install -g create-react-app
    • 切换到想要创建项目的目录，键入create-react-app pro-name
    • 进入到项目的文件夹中：cd pro-name
    • 启动项目：npm start

其中，pro-name是自定义的项目名。以上是在cmd中运行的指令，然后在浏览器的localhost:3030中可以看到效果。

另外，在vscode中，可以在终端中新建终端，输入npm start指令，或者使用快捷键Ctrl + Esc下面的按键。再者，推荐使用npm install yarn -g安装yarn包管理器，即可使用yarn start启动项目

JSX语法

  • 全称：JavaScript XML

  • react定义的一种类似于XML的JS扩展语法：JS + XML

  • 本质是React.createElement(component,props,...children)方法的语法糖

  • 作用：用来简化创建虚拟DOM

    • 写法：var ele = <h1> Hello JSX! </h1>
    • 注意1：它不是字符串，也不是HTML/XML标签
    • 注意2：它最终产生的就是一个JS对象

  • 标签名任意：HTML标签或其他标签

  • 注释语法是{/**/}。外层大括号表明要写JS语法，然后再用JS的注释语法即可

  • 编写标签时需注意：

    • 标签中混入了JS表达式，则需要用{}引入。
    • 类名指定不能用class，而要用className。
    • 内联样式，要用style={{key:value}}的形式，连字符的属性要用驼峰形式。（第一个大括号意即要用JS表达式，第二个大括号则是传入一个对象）
    • 定义虚拟DOM时，不要写引号，只能有一个根标签。
    • 关于标签的首字母
      • 若以小写字母开头，则将该标签转换为HTML中同名元素。若HTML中无对应元素，则报错
      • 若大写字母开头，react会去渲染对应的组件。若组件未定义，则报错

区分JS表达式和JS语句（代码）

  • 表达式：一个表达式会产生一个值，可以放在任何一个需要值的地方。
    • a
    • a + b
    • demo(1) （函数调用表达式，没写返回值也会返回undifined）
    • array.map()
    • function test（）{} (函数定义表达式)
  • 语句（代码）
    • if () {}
    • for () {}
    • switch () {case: ×××}
